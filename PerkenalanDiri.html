ASSALAMUALAIKUM WR. WB.

Perkenalkan nama saya Sri Wahyuni dan biasa dipanggil saya Yuni. Saya lahir di Kota Jember, 27 April 1999. Saat ini saya berumur 20 tahun, alamat saya di Jalan Rembangan, Dusun Rayap RT 02 RW 09, Kemuning Lor, Arjasa, Jember. Saat ini saya masih berkuliah di Politeknik Negeri Jember di jurusan Teknologi Informasi dan dari program studi Teknik Informatika. Sekarang saya sudah berada disemester 3 dan saya membuat website ini dalam rangka memenuhi tugas pertama saya yaitu mata kuliah Workshop Sistem Informasi Berbasis Web.

Saya sangat berterima kasih kepada bapak dosen yang telah membimbing saya dan teman - teman dalam mengenalkan dan mengajarkan kami untuk bisa membuat website seperti ini. Atas bimbingan dan materi yang disampaikan kepada kami yang memberikan pemahaman serta link untuk kita belajar dan mengaplikasikannya. Berikut link HTML dan video yang bermanfaat bagi kalian yang ingin belajar membuat website.

Pada awalnya saya tidak pernah berfikir untuk berkecimpung ataupun masuk di jurusan yang saat ini saya jalani. Saya merasa tidak memiliki passion di bidang Teknologi Informasi. Sebelum saya juga merasa bahwa saya salah jurusan karena saya dari lulusan SMA yang tidak pernah diperkenalkan lebih jauh tentang komputer dan yang saya tau hanya beberapa Microsoft Office seperti Word, Power Point, Excel. Sedangkan software yang sangat lain sebagainya hanya saya tau beberapa tapi tidak mengerti bagaimana cara penggunaannya. Akan tetapi saat ini saya sangat bersyukur bisa masuk di jurusan ini yang awalnya saya asal memilih sebuah jurusan.

Saya sangat berterima kasih kepada semua dosen yang telah membimbing saya selama ini sampai saya menginjak disemester 3 saat ini. Saya tidak menyangka bisa menjalaninya sampai sekarang, saya sangat berharap rasa penasaran saya tentang IT ini terus berkelanjutan supaya saat tidak pernah puas akan beberapa ilmu yang saya dapatkan. Demikian yang dapat saya sampaikan, kurang lebihnya mohon maaf.
